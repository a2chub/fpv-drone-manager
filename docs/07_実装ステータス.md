## データ構造

### Firestore コレクション

```
firestore/
├── users/{userId}
│   ├── email, displayName, photoURL, role
│   ├── profile { bio, location, socialLinks }
│   ├── settings {
│   │     defaultVisibility, emailNotifications, themeMode,
│   │     isProfilePublic, showEventHistory, commentApprovalRequired
│   │   }
│   ├── followingCount, followersCount  ← Phase 11で追加
│   ├── createdAt, updatedAt
│   │
│   ├── drones/{droneId}
│   │   ├── name, description, mainImageUrl, images[]
│   │   ├── category, specifications, status, isPublic
│   │   ├── createdAt, updatedAt
│   │   │
│   │   ├── parts/{partId}
│   │   │   ├── category, name, manufacturer, model
│   │   │   ├── purchasePrice, purchaseStore, purchaseDate, purchaseUrl
│   │   │   ├── images[], notes, isPublic
│   │   │   ├── createdAt, updatedAt
│   │   │   │
│   │   │   └── histories/{historyId}
│   │   │       ├── type, title, description, date
│   │   │       ├── images[], isPublic, createdAt
│   │   │
│   │   └── raceLinks/{linkId}
│   │       ├── raceId, raceName, raceDate, rank
│   │
│   ├── races/{raceId}
│   │   ├── name, date, location, officialUrl
│   │   ├── category, result, usedDroneId, usedDroneName
│   │   ├── impressions, images[], isPublic
│   │   ├── createdAt, updatedAt
│   │
│   ├── following/{targetUserId}  ← Phase 11で追加
│   │   └── followedAt
│   │
│   ├── followers/{followerUserId}  ← Phase 11で追加
│   │   └── followedAt
│   │
│   └── profileComments/{commentId}  ← Phase 11で追加
│       ├── authorId, authorName, authorPhotoURL
│       ├── content, isDeleted, isApproved
│       ├── createdAt, updatedAt
│
├── publicProfiles/{userId}
│   ├── displayName, photoURL, bio
│   ├── droneCount, raceCount
│   ├── featuredDrones[], recentRaces[]
│   ├── updatedAt
│
└── raceEvents/{eventId}  ← Phase 9で追加
    ├── organizerId, title, description
    ├── date, location, category, visibility
    ├── maxParticipants, registrationDeadline
    ├── createdAt, updatedAt
    │
    ├── participants/{participantId}
    │   ├── userId, displayName, status
    │   ├── droneId, droneName, isPublic
    │   ├── registeredAt, approvedAt
    │
    └── posts/{postId}
        ├── authorId, content, mediaUrls[]
        ├── createdAt
        │
        └── mediaComments/{commentId}
            ├── authorId, content, createdAt
```
